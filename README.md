# Taste Skill

The Anti-Slop Frontend Framework for AI Agents.

Open-source skill files that stop Cursor, Claude Code, Codex, Gemini CLI, v0, Lovable, and more from generating generic frontends.

## Structure

```
├── index.html          # Clean static HTML (no Next.js/RSC)
├── css/app.css         # Compiled Tailwind + custom animations + @font-face
├── js/app.js           # Vanilla JS: 3D carousel, scroll reveals, image error handler
├── fonts/              # 20 renamed woff2 files (Manrope, Playfair Display, IBM Plex Mono)
└── images/             # AVIF + SVG assets
```
