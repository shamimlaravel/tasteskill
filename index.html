Pick the right skill for the job.

taste-skill
v2 experimental

Reads the brief, infers the right design direction, and ships interfaces that do not look templated. Real design systems when applicable, audit-first on redesigns, strict pre-flight check. Install name: design-taste-frontend.

taste-skill-v1
Legacy

The original v1 of taste-skill, preserved for projects depending on its exact behavior. Install with --skill design-taste-frontend-v1. Use only if v2 breaks something specific.

gpt-tasteskill
GPT/Codex

Stricter variant for GPT and Codex models with stronger layout variance and motion direction.

image-to-code-skill

Generates design references first, analyzes them deeply, then implements the frontend closely.

redesign-skill

v2 (experimental)

What is new in v2.

A substantial 2026 rewrite of the default skill. The agent now reads the brief, infers the right direction, and ships interfaces that do not look templated. Same install command, much stricter ruleset.

Stability: v2 (experimental) is the new default. It is genuinely better than v1 and we recommend it, but it is still iterating. The install name (design-taste-frontend) stays stable. Rule wording may change before v2.0.0 stable cuts.

Existing users automatically get v2 on next install. Pinned to v1? use --skill design-taste-frontend-v1.
